## Welcome!
This is the course page for BIS 015L, **Data Science for Biologists** at UC Davis.
BIS015L is a course that introduces the fundamentals of data science, using the R software
environment. It is interpreted as a beginner course for coding. No prior coding experience
 is required to be able to enroll into the course. That is why the purpose of BIS015L is 
 to gain experience to learn a new "language" the R language. 
 
 This course teaches the fundamentals of importing,cleaning, analyzing and visualizing
biological data. As well as learning how to stay organized and manage work using Github 
and Github Desktop.
